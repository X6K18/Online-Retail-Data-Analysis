# Machine Learning Models - Online Retail Analytics

## Project Overview

This folder contains machine learning models developed for the Online Retail Analytics project.  
The objective is to build predictive and segmentation models that help analyze customer behavior, purchasing patterns, and business performance.

The models are designed to support data-driven business decisions such as:

- Customer segmentation
- Sales prediction
- Customer value analysis
- Churn prediction
- Product recommendation
- Customer retention strategy
